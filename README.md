
# kingfisher

ERP For Taihuoniao.

###运行环境要求

* PHP >= 5.5.9
* OpenSSL PHP Extension
* PDO PHP Extension
* Mbstring PHP Extension
* Tokenizer PHP Extension

###安装使用

####第一步：安装composer包管理器

访问[composer](http://pkg.phpcomposer.com/)，根据文档说明安装composer。
    
####第二步：开发环境生成ssh公钥。

```
ssh-keygen -t rsa -C "your email!"
```

####第三步：克隆kingfisher代码

```
git clone git@github.com:purpen/kingfisher.git
```
